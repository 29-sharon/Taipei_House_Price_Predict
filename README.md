# 🏠 台北市房價預測模型

本專案使用 Python 建立兩種機器學習模型（**線性迴歸**與**隨機森林**）來預測台北市的房價（總價），並進行準確度比較。資料處理與模型訓練皆透過 `pandas` 與 `scikit-learn` 完成。

---

## 📌 功能特色

- 讀取台北市不動產交易資料進行分析。
- 建立 **線性迴歸** 與 **隨機森林** 兩種模型。
- 評估模型表現（使用 **RMSE** 和 **R² 分數**）。
- 輕鬆比較模型預測效果。
